# 방탈출 API

| Method | Endpoint             | Description  |
|--------|----------------------|--------------|
| GET    | `/admin`             | 어드민 페이지 요청   |
| GET    | `/admin/reservation` | 예약 관리 페이지 요청 |
| GET    | `/reservations`      | 예약 정보        |
| POST   | `/reservations`      | 예약 추가        |
| DELETE | `/reservations/{id}` | 예약 취소        |
